## 🚀 Proyectos Activos

A continuación, se detalla el listado de cada uno de los proyectos, su propósito y recursos correspondientes:

<br>

### 📈 [ds-qs](./ds-qs)

> **ERP Intelligent Suite (Phase 1)**

- **📓 Descripción:** ERP de siguiente generación diseñado para la automatización contable total. La arquitectura se basa en la potencia de Velneo para la gestión de datos e interfaz, potenciada por un ecosistema de microservicios en Node.js para la clasificación contable automática mediante IA.
- **🔗 Repositorio:** [https://github.com/arturomiranda/ds-qs.git](https://github.com/arturomiranda/ds-qs.git)

<br>
<hr>
<br>


### 📦 [ds-api](./ds-api)

> **Motor de Facturación Electrónica y API REST (CFDI 4.0)**

- **📓 Descripción:** Motor de servicios backend de alto rendimiento enfocado en el ciclo completo de facturación electrónica mexicana. Contiene lógica para lectura de certificados CSD, sellado digital, timbrado y distribución por SMTP. Se encuentra altamente optimizado para ejecutarse sobre Node.js 22 en WSL (Ubuntu).
- **🔗 Repositorio:** [https://github.com/arturomiranda/ds-api.git](https://github.com/arturomiranda/ds-api.git)

<br>
<hr>
<br>

### 🔑 [lms-app](./lms-app)

> **License Management System (LMS)**

- **📓 Descripción:** Sistema centralizado para la administración, generación y control de licencias de software, usuarios y recursos de infraestructura.
- **🔗 Repositorio:** [https://github.com/arturomiranda/lms-app.git](https://github.com/arturomiranda/lms-app.git)

### 🛒 [web-app](./web-app)

> **Plataforma Web Corporativa y de E-commerce**

- **📓 Descripción:** Es una solución web integral diseñada para ofrecer una sólida presencia corporativa pública combinada con un robusto sistema de ventas en línea. Adicionalmente, incluye un portal protegido adaptado para la gestión de relaciones tanto con clientes como con distribuidores (compras al por mayor y reportes).
- **🔗 Repositorio:** [https://github.com/arturomiranda/web-app.git](https://github.com/arturomiranda/web-app.git)
